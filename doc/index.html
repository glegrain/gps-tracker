GET /api/devices

Retrieves all registered devices.

Examples:

GET /api/devices

Response:

The response will be a JSON array of objects. Each object will
contain an individual device, for example:

[
 {
   "device_id": 1,
   "name": "gps1",
   "description": "this is a description",
   "date_added": "2014-01-16T23:01:12.132Z"
 },

     ...

 {
   "device_id": 4,
   "name": "gps4",
   "description": "this is a bike gps",
   "date_added": "2014-01-16T23:01:13.373Z"
  }
]

Errors:

  • 500 with a database error

GET /api/coordinates/:id

Retrieves device position.

Examples:

GET /api/coordinates/1

Response:

Returns the coordinates associated with a device as an object, as in:

{
  "id": 2,
  "device_id": 1,
  "latitude": 10,
  "longitude": 10,
  "timestamp": "2014-01-16T23:44:18.969Z"
}

  • 500 with a database error
